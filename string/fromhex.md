# 从16进制转回(string.fromHex)

## 函数

```lua
string.fromHex(字符串)
```

## 参数

| 参数  | 类型     | 必填 | 说明      |
| --- | ------ | -- | ------- |
| str | string | 是  | 待处理的字符串 |

## 返回值

| 返回值 | 类型     | 说明     |
| --- | ------ | ------ |
| res | string | 处理过的文本 |

## 示例

```lua
string.toHex(" 随便写--个字符 "):fromHex() -- 字符串没有任何变化
```
