# 逐字分割文本(string.chars)

## 函数

```lua
string.chars(字符串)
```

## 参数

| 参数  | 类型     | 必填 | 说明      |
| --- | ------ | -- | ------- |
| str | string | 是  | 待处理的字符串 |

## 返回值

| 返回值 | 类型    | 说明         |
| --- | ----- | ---------- |
| res | table | 分割后的字符串 数组 |

## 示例

```lua
string.split("随便写--个字符","--") -- 这将返回包含每个字的数组 
```
