# 是否是整数(string.isInteger)

## 函数

```lua
string.isInteger(字符串)
```

## 参数

| 参数  | 类型     | 必填 | 说明      |
| --- | ------ | -- | ------- |
| str | string | 是  | 待处理的字符串 |

## 返回值

| 返回值 | 类型   | 说明    |
| --- | ---- | ----- |
| res | bool | 是否为整数 |

## 示例

```lua
string.isInteger("1") -- 返回true
```
