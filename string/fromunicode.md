# unicode到 UTF8(string.fromUnicode)

## 函数

```lua
string.fromUnicode(字符串)
```

## 参数

| 参数  | 类型     | 必填 | 说明      |
| --- | ------ | -- | ------- |
| str | string | 是  | 待处理的字符串 |

## 返回值

| 返回值 | 类型     | 说明     |
| --- | ------ | ------ |
| res | string | 处理过的文本 |

## 示例

```lua
string.fromUnicode("一个unicode字符串") 
```
