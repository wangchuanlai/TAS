# 是否是小写字母(string.isLower)

## 函数

```lua
string.isLower(字符串)
```

## 参数

| 参数  | 类型     | 必填 | 说明      |
| --- | ------ | -- | ------- |
| str | string | 是  | 待处理的字符串 |

## 返回值

| 返回值 | 类型   | 说明                |
| --- | ---- | ----------------- |
| res | bool | 是返回true 否则返回false |

## 示例

```lua
string.isLower("1") -- 返回false
```
