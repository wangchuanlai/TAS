# 分割文本(string.split)

## 函数

```lua
string.split(字符串, 分隔符)
```

## 参数

| 参数    | 类型     | 必填 | 说明      |
| ----- | ------ | -- | ------- |
| str   | string | 是  | 待处理的字符串 |
| split | string | 是  | 分隔符     |

## 返回值

| 返回值 | 类型    | 说明         |
| --- | ----- | ---------- |
| res | table | 分割后的字符串 数组 |

## 示例

```lua
string.split("随便写--个字符" ,"--") -- 这将返回
```
