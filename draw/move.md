# 移动(obj:move())

## 对象方法

```lua
obj:move(坐标x, 坐标y)
```

## 参数

| 参数 | 类型  | 必填 | 说明  |
| -- | --- | -- | --- |
| x  | int | 是  | 横坐标 |
| y  | int | 是  | 纵坐标 |

## 返回值

| 返回值 | 类型       | 说明     |
| --- | -------- | ------ |
| obj | userdata | 屏幕绘制对象 |

## 示例

```lua
--在屏幕 100,200 的位置创建一个 宽度100 高度100的 屏幕绘制对象  
local sd = screenDraw.init(100,200,100,100)
sd:show()  --在屏幕上显示
sys.msleep(1000)
sd:move(200,200)  --移动到 200,200的位置
```
