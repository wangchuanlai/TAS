# 触摸移动(obj:move)

## 对象方法

```lua
obj:move(屏幕坐标x,屏幕坐标y)
```

## 参数

| 参数 | 类型  | 必填 | 说明    |
| -- | --- | -- | ----- |
| x  | int | 是  | 屏幕坐标x |
| y  | int | 是  | 屏幕坐标y |

## 返回值

| 返回值 | 类型       | 说明      |
| --- | -------- | ------- |
| obj | userdata | touch对象 |

## 示例

```lua
--简单示例
local t = touch.down(100,100 ) --手指按下
t:move(200,200)        --手指移动到 200-200的位置
t:up()    --手指抬起

--链式调用示例
touch.down(100,100):move(200,200):up()    --和上方示例 同样的效果

```
