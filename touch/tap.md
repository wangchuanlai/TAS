# 触摸一次屏幕(touch.tap)

## 函数

```lua
touch.tap(屏幕x ,屏幕y ,[手指抬起时间], [压力])
```

## 参数

| 参数    | 类型  | 必填 | 说明             |
| ----- | --- | -- | -------------- |
| x     | int | 是  | 屏幕坐标 x         |
| y     | int | 是  | 屏幕坐标 y         |
| sleep | int | 可选 | 手指抬起时间 默认 30毫秒 |
| press | int | 可选 | 施加压力 默认100     |

## 无返回值

## 示例

```lua
touch.tap(100,100)    --触摸屏幕 100-100位置

touch.tap(100,100 ,50 ,200)    --触摸屏幕 100 -100 的位置 抬起时间为50毫秒 并施加压力 200
```
