# 触摸屏幕按下(touch.down)

## 函数

```lua
touch.down(屏幕坐标x , 屏幕坐标y ,[压力])
```

## 参数

| 参数    | 类型  | 必填 | 说明        |
| ----- | --- | -- | --------- |
| x     | int | 是  | 屏幕坐标x     |
| y     | int | 是  | 屏幕坐标y     |
| press | int | 可选 | 压力 默认 100 |

## 返回值

| 返回值 | 类型       | 说明   |
| --- | -------- | ---- |
| obj | userdata | 触摸对象 |

## 示例

```lua
local t = touch.down(100,100 ) --手指按下
t:move(200,200)        --手指移动到 200-200的位置
t:up()    --手指抬起
```
