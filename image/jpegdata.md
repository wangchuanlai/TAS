# 获取图片的JPEG数据(imgObj:jpegData)

## 对象方法

```lua
imgObj:jpegData()
```

## 无参数

## 返回值

<table><thead><tr><th>返回值</th><th width="135.33333333333331">类型</th><th>说明</th></tr></thead><tbody><tr><td>data</td><td>string</td><td>图片数据</td></tr></tbody></table>

## 示例

```lua
--简单的示例
screen.image():jpegData()  --截图并读取JPEG数据

```
