# 图片右旋转90度(imgObj:turnRight)

## 对象方法

```lua
imgObj:turnRight()
```

## 无参数

## 返回值

<table><thead><tr><th>返回值</th><th width="135.33333333333331">类型</th><th>说明</th></tr></thead><tbody><tr><td>imgObj</td><td>userdata</td><td>新的图片对象</td></tr></tbody></table>

## 示例

```lua
--简单的示例
screen.image():turnRight()  --把图片向右旋转90度

```
