# 设置设备名称(device.setName)

## 函数

```lua
device.setName(新的设备名称)
```

## 参数

| 参数   | 类型     | 必填 | 说明     |
| ---- | ------ | -- | ------ |
| name | string | 是  | 新的设备名称 |

## 返回值

| 返回值    | 类型   | 说明                    |
| ------ | ---- | --------------------- |
| result | bool | 成功返回 true  失败返回 false |

## 示例

```lua
device.setName("设置一个新的设备名称")
```
