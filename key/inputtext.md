# 输入文字(key.inputText)

## 函数

```lua
key.inputText(字符串)
```

## 参数

| 参数  | 类型     | 必填 | 说明  |
| --- | ------ | -- | --- |
| str | string | 是  | 按键码 |

## 无返回值

## 示例

```lua
key.inputText("随便输入几个文字") --模拟粘贴字符串
```
