# 提示窗口(sys.alert)(废弃)

## 函数

```lua
sys.alert(提示信息,[显示时间])
```

## 参数

| 参数       | 类型     | 必填 | 说明                       |
| -------- | ------ | -- | ------------------------ |
| message  | string | 是  | 提示文本                     |
| showTime | int    | 可选 | 显示的时间 参数为0时 将一直显示到手动操作为止 |

## 无返回值

## 示例

```lua
sys.alert("弹出一个窗口" ,0)
```
